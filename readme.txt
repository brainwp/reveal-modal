Add cool modal to your site, open posts with ajax in all pages!

== Description ==

Reveal Modal WP is a plugin based on foundation reveal modal JS, a solution that will allow you to set up a cool modal window to any of your posts and pages. It's a simple and easy solution that will rock your website!

After installing it, you will see an option at bottom of posts/pages, just select to open in modal window. Yes, it's that easy! I told you...

If you have any problems with the plugin after reading the FAQ, Other Notes, etc. Please visit the [repository at github](https://github.com/brasadesign/reveal-modal "Reveal Modal WP") and open an issue.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugin/reveal-modal.zip` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Reveal Modal screen to configure the plugin
